EMFJSON is a JSON mapper for [EMF](http://www.eclipse.org/emf).

Current version is 0.3.2.

Eclipse Update Site (p2 repository): http://ghillairet.github.com/emfjson/releases/0.3.2/

# Dependencies

* EMF 2.7.0 or later
* Jackson 1.6.0 or later
